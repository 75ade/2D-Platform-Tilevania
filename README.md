# 2D-Platform-Tilevania

Hello there, I would like to introduce to you about a 2D platform game created using Unity 6.0. 
This project was developed based on a game development course from Udemy and it was further improved with additional features implemented by me.

## 🎮 Gameplay
- Control a character (Bronzy or Reddy) to explore and escape from a dungeon
- Try to survive and manage your health by avoiding or shooting the enemies (Gooba) down 
- Achieve score by collecting coins

## 🛠️ Features
- Player movement (Keyboard)
- Shooting system (Mouse or keyboard)
- Basic enemy movement
- Collision handling
- SFX
- Basic animation

## 💻 Technologies used
- Unity (version: 6.3)
- Progamming language: C#
- Unity Input System
- Git & Github
